# Project Instructions for Claude

## Project Overview
This is the Inerci AI website - a Lithuanian AI automation agency landing page with an internal admin dashboard.

## Structure
- `/src/app/` - Next.js App Router pages
- `/src/app/admin/` - Internal Audit & Proposal Copilot (separate from landing page)
- `/src/components/socio/` - Landing page components
- `/docs/prd.md` - PRD for the admin app

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- Cal.com for booking integration

## Coding Standards
- Use TypeScript strictly
- Prefer functional components with hooks
- Use Tailwind for styling (no CSS modules)
- Keep components small and focused
